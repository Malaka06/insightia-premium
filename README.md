# InsightIA — Voice of Customer Decision Intelligence

Plateforme Streamlit premium pour transformer des verbatims clients en backlog priorisé.

## Lancer en local

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

## Fonctionnalités

- 3 démos : E-commerce, SaaS B2B, Ambassade / Services consulaires du Congo
- Import CSV utilisateur
- Détection automatique des colonnes texte, date, canal, score
- Taxonomie métier explicable
- Sentiment simple par mots-clés
- Scoring criticité / fréquence / blocage
- Backlog priorisé P0 / P1 / P2
- Executive View, Explorer, Prioritization Board, Reports
